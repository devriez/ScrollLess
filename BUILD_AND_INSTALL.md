# Сборка и установка приложения

## Что нужно установить на компьютер

1. Установи Android Studio: https://developer.android.com/studio
2. При первом запуске Android Studio установи Android SDK, Platform Tools и Android SDK Build-Tools.
3. Открой этот проект в Android Studio.
4. Дождись Gradle Sync. Android Studio сама скачает Gradle-плагины и зависимости проекта.

## Подготовка телефона

1. Открой настройки телефона.
2. Перейди в раздел `О телефоне`.
3. Нажми 7 раз на `Номер сборки`, чтобы включить режим разработчика.
4. Перейди в `Настройки -> Для разработчиков`.
5. Включи `Отладка по USB`.
6. Подключи телефон к компьютеру USB-кабелем.
7. На телефоне подтверди разрешение на USB-отладку.

## Быстрый запуск из Android Studio

1. В Android Studio выбери телефон в списке устройств.
2. Нажми `Run`.
3. Приложение соберется и установится на телефон.

После установки открой приложение и включи:

1. `Accessibility` в системных настройках.
2. При необходимости `Overlay`-разрешение.

## Сборка APK через терминал

Если в проекте есть Gradle Wrapper:

```bash
./gradlew assembleDebug
```

Если Gradle Wrapper еще не создан, открой проект в Android Studio и выполни сборку оттуда. Затем можно добавить wrapper командой:

```bash
gradle wrapper
```

Debug APK будет лежать здесь:

```text
app/build/outputs/apk/debug/app-debug.apk
```

## Установка APK на телефон через ADB

Проверь, что телефон виден компьютеру:

```bash
adb devices
```

Установи debug APK:

```bash
adb install -r app/build/outputs/apk/debug/app-debug.apk
```

Если раньше была установлена старая версия с пакетом `com.devriez.blackwhite`, удали ее один раз:

```bash
adb uninstall com.devriez.blackwhite
```

## Включение полного черно-белого режима

Полный режим требует разовой выдачи системного разрешения через ADB:

```bash
adb shell pm grant com.devriez.scrollless android.permission.WRITE_SECURE_SETTINGS
```

После этого:

1. Открой приложение.
2. Выбери режим `Полный`.
3. Выбери приложения, для которых нужно включать grayscale.
4. Включи Accessibility Service для приложения.

## Проверка быстрого режима

Быстрый режим не требует ADB-разрешения `WRITE_SECURE_SETTINGS`.

1. Открой приложение.
2. Выбери режим `Быстрый`.
3. Выбери до двух приложений в Free-версии.
4. Включи Accessibility Service.
5. Открой выбранное приложение.

На экране должен появиться серый визуальный фильтр.

## Free и Pro

В текущей MVP-версии Pro включается локальным переключателем внутри приложения, чтобы можно было тестировать лимиты без Google Play.

- Free: максимум 2 выбранных приложения.
- Pro: без лимита, плюс пауза, расписание, профили и быстрый toggle.

Перед публикацией в Google Play локальный Pro-переключатель нужно заменить на Google Play Billing.
